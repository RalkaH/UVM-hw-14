UVM-hw-14 — ассемблер для учебной виртуальной машины (УВМ), вариант №14.

Назначение
CLI-приложение читает текст программы на языке ассемблера, строит промежуточное представление (IR) и формирует машинный код (двоичный файл) согласно спецификации УВМ.
В режиме тестирования выводит IR в формате полей и значений.

Язык ассемблера
Одна инструкция на строку.
Аргументы — десятичные целые числа.
Разделитель аргументов — запятая.

Поддерживаемые инструкции
LOAD A, B, C
Загрузка константы.
A = 82
B = адрес
C = константа

READ A, B, C, D
Чтение значения из памяти.
A = 25
B = адрес
C = адрес
D = смещение

WRITE A, B, C, D
Запись значения в память.
A = 57
B = адрес
C = адрес
D = смещение

UMINUS A, B, C
Унарная операция: унарный минус.
A = 33
B = адрес
C = адрес

Структура проекта
assembler.py — ассемблер (парсинг, IR, генерация байт)
grammar.lark — грамматика языка ассемблера (Lark)
examples/ — тестовые программы из спецификации
tests/test_assembler.py — pytest-тесты на совпадение байтов с примерами из спецификации
requirements.txt — зависимости

Установка
pip install -r requirements.txt

Запуск
Сборка бинарника:
python assembler.py -i examples/test1.s -o out.bin

Режим тестирования (печать IR):
python assembler.py -i examples/test1.s -o out.bin --test

Тесты
pytest -v
